# Cardinalidad
Tags: #LFyC
Fecha de Creación: 05-03-2026
# Nota
---
El concepto de cardinalidad viene primero de conjuntos finitos, que es la cantidad de elementos de un conjunto, pero en conjuntos infinitos, se piensa más como para comparar el "tamaño" entre conjuntos
Fue pensado por Cantor, en una época donde se pensaba que el infinito sólo existe *en potencia*, sólo podemos construir conjuntos finitos de forma infinita hasta conseguir nuestro elemento. Cantor lo meditó como que el infinito existe *en el acto*, donde es posible insertar todos los elementos de un conjunto infinito en una "bolsa"
Se puede pensar como el *algoritmo eficiente de chequeo de igualdad de listas*, donde se recorre elemento por elemento para ver si hay alguna relación uno a uno por elemento. Si se llega al fin de un conjunto pero no para el otro, tienen largos distintos. Sino, tienen largos iguales. De allí nace la idea de la relación uno a uno entre conjuntos

# Def. de un conjunto inductivo y sus usos
## La idea en una oración
---
> Para los conjuntos numerables o universales, se puede definir a partir de unas reglas base y de inducción para armar objetos del conjunto
## Desarrollo
---
A veces, requerimos de trabajar sobre conjuntos infinitos de manera finita, y para eso tenemos muchas estrategias. Una de ellas es describir el conjunto de forma inductiva
Esto se puede hacer en conjuntos numerables o universales, ya que necesitamos que las reglas se aplique para todos los elementos, y que se puedan enlistar uno tras otro usando las reglas
$$
\begin{align*}
	\text{Ejemplo: } \mathbb{N} \\
	\mathbb{N} \text{ es conjunto mínimo tal que:} \\
	\text{1. } 0 \in \mathbb{N} \\
	\text{2. } n \in \mathbb{N} \Rightarrow n+1 \in \mathbb{N} \\
\end{align*}
$$
*Es esencial que es el mínimo conjunto tal que cumple con las reglas, para evitar cosiderar elementos que no pertenecen*
De esta forma, viene casi que gratis el *PIP* o el Principio de Inducción Primitiva, que nos permite demostrar propiedades de forma inductiva en el conjunto

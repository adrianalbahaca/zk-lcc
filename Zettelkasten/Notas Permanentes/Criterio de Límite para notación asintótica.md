# Nota
---
Sean dos funciones $f, \ g: \mathbb{N} \rightarrow \mathbb{R}$, es posible determinar su notación asintótica calculando el límite del cociente
$L = \lim_{n \to \infty} f(n)/g(n)$ 
 - Si tenemos que $L = 0$, entonces $f(n) \in O(n), f(n) \notin \Omega(n)$
 - Si $0 < L < \infty$, entonces $f(n) \in \Theta(n)$
 - Si $L = \infty$, entonces $f(n) \in \Omega(n), f(n) \notin O(n)$ 
 Esto se puede usar en conjunto con las definiciones de las notaciones individuales, en caso de que el límite del cociente salga fácil

# Def. y Ej. de Espacio Muestral
## La idea en una oración
---
> El espacio muestral es el conjunto de todos los resultados posibles en un experimento
## Desarrollo
---
Considerando una baraja de cartas, quiero saber qué chances tengo de obtener una Q de Picas. Para esto, primero tengo que definir todas las cartas que puedo obtener

Este es el *Espacio muestral*, el conjunto de todos los resultados posibles de un experimento. Para este ejemplo, se puede ver como 
$$C = \{(A,Corazon), (2, Corazon), ..., (K, Corazon), (A, Picas), ... \}$$
De esta forma, podemos después ver cuál *suceso* puede ocurrir y cuál es su probabilidad
Este conjunto puede ser finito, infinito numerable o infinito no numberable

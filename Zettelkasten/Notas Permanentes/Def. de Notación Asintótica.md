# Nota
---
La notación asintótica permite analizar la eficiencia de un algoritmo acorde a qué tan grande sea su entrada. Se suele definir una función de la siguiente manera
$T: \mathbb{N} -> \mathbb{R}$ 
Esta función es el tiempo de ejecución en el peor de los casos acorde a una entrada de un cierto tamaño $n$ 
Para tener una idea intuitiva de esto, se suele usar 3 notaciones principales: $O, \Omega, \Theta$ 
Toma en cuenta que T suele ser *cerrada*. Es decir, no se puede expresar en términos de sí misma
Usualmente, se puede derivar sus *órdenes de crecimiento* haciendo operaciones, o tanteando con inducción

# Nota
---
> Sean $f, \; g: \mathbb{N} -> \mathbb{R}$, se dice que $f \in O(g)$ Si existe $c \in \mathbb{R}^+$, $n_0 \in \mathbb{N}$  constantes tal que $\forall n > n_0, 0 \le f(n) \le c \times g(n)$  

Quiere decir que, con tal $c$, es posible acotar superiormente a la primera función. Esto no da una idea general de cuál puede ser el peor caso

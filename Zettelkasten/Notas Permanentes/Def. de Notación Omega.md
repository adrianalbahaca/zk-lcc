# Nota
---
> Sean $f, \; g: \mathbb{N} -> \mathbb{R}$, se dice que que $f \in \Omega (g)$ si existe $c \in \mathbb{R}^+$, $n_0 \in \mathbb{N}$ constantes tal que $\forall n > n_0, \; 0 \le c \times g(n) \le f(n)$ 

Esta notación acota inferiormente a una función. Esto puede determinar el mejor caso posible

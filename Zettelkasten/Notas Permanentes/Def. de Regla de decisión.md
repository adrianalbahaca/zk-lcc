# Nota
---
La Regla de la Decisión es una métrica que se define antes del estudio para determinar si se rechaza $H_0$ o no. Esta se basa en el *nivel de significancia*, tal que, si $p-value <= \alpha$, se rechaza $H_0$ y se afirma que la respuesta es estadísticamente significativa. Sino, no se rechaza $H_0$ y se dice que la no es estadísticamente significativo

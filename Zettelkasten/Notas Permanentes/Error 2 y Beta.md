# Def. de Error 2 y su utilidad
## La idea en una oración
---
> El error 2, o $e_2$, es aquel que pasa al no negar $H_0$, cuando la realidad sustenta a $H_1$. $\beta$ se calcula en una gráfica discreta como el área bajo la curva de $H_1$ en toda la zona de no rechazo
## Desarrollo
---
El error 2, o $e_2$ es el error cometido al no rechazar a $H_0$ cuando la realidad sustenta a $H_1$. Está relacionado con la *Potencia*

Este error se puede minimizar aumentando $\alpha$ pero eso implicaría un test más flexible

Su probabilidad asociada, o $\beta$, se calcula como el área bajo la curva de $H_1$ desde $X_{critico}$ por debajo de toda la zona de no rechazo, y se anota como $P(e_2)$ 

En realidad, este error no se puede calcular si no se tiene un gráfico discreto, o si no se tiene una regla de decisión

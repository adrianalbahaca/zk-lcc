# Nota
---
La estadística Bayessiana tiene una perspectiva distinta. Después de crear el modelo usando una base de datos de entrenamiento, al tener datos nuevos, *se considera el modelo a base de los datos*. Es decir, si aparece un dato que, a primeras, puede ser inusual, entonces se cuestiona el modelo en sí, por lo menos para asegurarse antes de descartar el dato. De esta manera, el modelo puede ser actualizado frecuentemente

Esta es una rama que no será vista en la materia, pero que puede ser investigada por mi lado

# Nota
---
La estadística clásica se basa en una interpretación *frecuentista*. Basándose en una cierta cantidad de datos, se crea un modelo *fijo* que los interpreta lo más cercano posible. Al insertar datos nuevos, se analiza los datos *a base del modelo*. Es decir, el modelo se mantiene fijo, considerando datos inusuales como datos que, como no coinciden en el modelo, no son válidos

Esta será la rama de enfoque

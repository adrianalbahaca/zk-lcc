# IPC y métodos más comunes
## La idea en una oración
---
> Hay múltiples formas de comunicación entre procesos. Algunos son Señales, Pipes, Sockets, y más
## Desarrollo
---
Mientras dos procesos hacen su trabajo a la vez, puede que sea necesario transmitir información entre los dos para colaborar entre sí. Esto se llama *IPC* o *Inter-Communication Process*, y hay múltiples interfaces para hacer esto

- Señales
- Pipes
- Sockets
- Message Queues
- Shared Memory
- Remote Procedure Calls

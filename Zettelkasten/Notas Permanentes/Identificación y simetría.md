# Tipos de comunicación
## La idea en una oración
---
> Las computadoras se pueden comunicar de manera *directa*, *indirecta*, *simétrica* o *asimétrica*
## Desarrollo
---
La comunicación directa pide que el emisor explicite el destinatario remitente a la comunicación de forma específica, bueno para mensajes específicos pero algo tedioso si se tiene que escribir quién lo debe recibir. 

La comunicación indirecta envía y recibe mensajes a través de un buffer, lo cual es bueno para mensajes tipo *broadcast* pero esto permite que cualquiera se meta a ver el mensaje. Erlang usa comunicación directa

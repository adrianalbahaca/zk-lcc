# MPI: Librería para hacer computación distribuída
## La idea en una oración
---
> MPI usa sockets para ejecutar computación distribuída y paralela
## Desarrollo
---
Para C, existe *MPI*, que es una librería portátil para ejecutar computación paralela y distribuida. Proporciona la capacidad de paralelizar  código de forma *no homogénea*, como por ejemplo paralelizar un código en en una red de computadoras, o nodos, que se comunican en la misma red 

Bajo el capó, usa sockets para comunicarse entre las computadoras y enviar y recibir información entre los nodos de la red

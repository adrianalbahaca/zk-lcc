# PIP y su importancia 
## La idea en una oración
---
> El Principio de Inducción Primitiva nos permite asegurar que, si una propiedad cumple en el caso base y en el paso inductivo, cumple para todos los elementos del conjunto
## Desarrollo
---
Al definir un conjunto inductivo, tenemos como herramienta automática el demostrar propiedades en el conjunto de forma inductiva. Para una propiedad arbitraria, esto se conoce como el *Principio de Inducción Primitiva*
Para verlo en acción, considere el conjunto de los elementos naturales previamente definido, y definamos su *PIP*:
$$
\begin{align}
	\text{Sea P una propiedad tal que: } \\
	\text{1. } P(0) \text{ cumple} \\
	\text{2. } \forall n \in \mathbb{N}, P(n) \text{ cumple } \Rightarrow P(n+1) \text{ cumple} \\
	\text{Entonces } P \text{ cumple para todo } n \in \mathbb{N}
\end{align}
$$
De esta forma, si una propiedad cualquiera cumple estas reglas, cumple para todos los elementos de $\mathbb{N}$. Nota inmediatamente lo poderoso de la herramienta. Esto nos puede demostrar cosas como que un conjunto esté dentro del otro, y viceversa

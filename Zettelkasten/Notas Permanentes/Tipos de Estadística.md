# Tipos de Estadística
Tags: #PyE
Fecha de Creación: 04-03-2026
# Nota
---
Hay dos tipos principales de estadística a estudiar:
- ***Estadística descriptiva:*** Reune una cantidad de datos para simplemente describirlos. Compila los datos y los grafica de una forma fácil de ver para que las personas puedan interpretar los datos
- ***Estadística inferencial:*** Usando una muestra, se procesa para tratar de sacar conclusiones. Se trata de extrapolar las cosas con una muestra *representativa*
